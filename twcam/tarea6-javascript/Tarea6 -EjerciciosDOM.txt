1.7 	Desarrolla el código Javascript necesario para:

	- Controlar un mínimo/máximo de checkbox seleccionados (definir la función comprobar(evento)).
	
	- Pasar elementos entre distintos Selects (definir la función selects(opc)).
	
	- Cambiar el estilo de un campo obligatorio cuando no se ha rellenado y se ha hecho un submit. (definir la función comprobar_obligatorio())
	
	- Cambiar el color de un texto al pasar el cursor por encima (definir resaltar(obj) y desresaltar(obj)).
	
	- Asociar sliders de HTML5 con el modelo de cajas de CSS3 para un objeto ((definir la función cajas(elem, valor), donde elem será M, B o P (Margen, Borde o Padding)).
	
	- Crear una tabla tipo cebra (celdas alternativas de distintos colores) (definir función cambiafilaimpar(color) y función cambiafilapar(color))
	
	- Añadir Texto introducido en un input al cuerpo del documento. (definir funcion addTexto())
